This folder contains two sets of images divided into 2 classes: trees
and balls.
The training set contains 20 images for each class, the test set 10 images for each class.
Images are 16x15 pixels large and saved in PGM ASCII format.

The image format is as follows:

2 line head (can be ignored)
width height  (format of the picture in ascii)
pixel values as numbers between 0 and 255
